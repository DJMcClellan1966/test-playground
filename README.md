## Getting Started

1. **Share an idea** - Describe what you want to build or explore
2. **We plan together** - I'll help break it down into steps
3. **Build iteratively** - We code, run, and refine
4. **Learn and document** - Save what works for future reference

## Tips for Working with AI

- **Be specific** about what you want to achieve
- **Share context** about why you're interested in an idea
- **Ask questions** if something isn't clear
- **Iterate freely** - first versions don't need to be perfect
