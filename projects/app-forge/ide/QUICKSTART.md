# IDE quickstart

## Backend (FastAPI)

```bash
cd projects/app-forge/ide
python backend.py
```

## Frontend

Open your browser to:

```
http://127.0.0.1:8000/static/index.html
```

- File browser: left
- Code editor: center
- Problems/terminal: bottom
- Agent chat: right

You can browse desktop folders, edit files, chat with the agent, and see problems/terminal output.
